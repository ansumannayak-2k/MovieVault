**MovieVault** is a clean and easy-to-use **Movie Search & Watchlist Web Application** built with HTML, CSS, and JavaScript using the **OMDb API**.  
You can search for movies, view details, and save your favorites to a Watchlist stored locally in your browser.

---

## Live Demo  
**Website:** https://movie-vault-d0336d.netlify.app/ 

---

## Features

1. Movie Search - Search movies in real time using OMDb API 
2. Watchlist - Save your favorite movies (stored in LocalStorage) 
3. Dark / Light Theme - Theme switch with saved preference 
4. Modern UI - Clean, responsive card-based UI 
5. Mobile Friendly - Works smoothly on all devices 

---

## Tech Stack

1. UI / Frontend - HTML, CSS, JavaScript 
2. API - OMDb API 
3. Data Storage - LocalStorage 
4. Deployment - Netlify
